# AIRWAVES - Premium Hemp Products E-Commerce

## Deploy via Netlify CLI

```bash
npm install -g netlify-cli
cd airwaves
netlify login
netlify link
netlify deploy --build --prod
```

After deploy, visit: https://airwaves3.netlify.app/api/db-init
